exceltomysql¶

Authors: Xiangyong Luo
Email: luoxiangyong01@gmail.com

exceltomysql is a tool to convert excel files (.xls, .xlsx, .csv) to mysql database.

**If you use exceltomysql in your research, please cite: https://github.com/xyluo25/exceltomysql **

Package dependencies¶

  • [X] sqlalchemy

  • [X] pandas
